# Syntax and Textures names
( A ) = ( Albedo ) ( Base Color ) ( Diffuse ) ( Color )

( E ) = ( Emission ) ( Emissive ) ( Glow ) ( Light )

( H ) = ( Height ) ( Displacement )

( M ) = ( Metallic ) ( Metalness )

( N ) = ( Normal )

( O ) = ( Ambient Occlusion ) ( Occlusion )

( R ) = ( Roughness )

( REF ) = ( Refraction )

( SPEC ) = ( Specular )

( SSS ) = ( Subsurface Scattering )

( T ) = ( Transparency )
